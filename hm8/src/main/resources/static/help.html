Запросы, которые доступны гостю и менеджеру:

http://localhost:3466/products/all - GET запрос, отдает все продукты

http://localhost:3466/products/allByCompany?name=companyName - GET запрос, отдает все продукты, которые были произведены компанией с именем companyName


Запросы, которые доступны только менеджеру:

http://localhost:3466/products/delete?name=productName - POST запрос, удаляет продукт с именем product name

http://localhost:3466/products/save (в body передать JSON: {"name": "productName", "companyName": "companyName", "count": 10}) - POST запрос, сохраняет продукт, с данными из Body
